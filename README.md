# ngrx-gen
A simple ngrx generator supporting RXJS version 5 and version 6

## How to

#### Install
$ npm install ngrx-gen -g

#### Run
- $ ngrx-gen

![Alt text](sample/ngrx-gen.png?raw=true "ngrx-gen")

- Select one of the option
- Enter service name
- Enter relative path of the folder
